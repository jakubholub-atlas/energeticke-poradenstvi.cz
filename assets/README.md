## Rychlé nasazení na GitHub Pages
1) Vytvoř repo (např. energeticke-poradenstvi.cz)
2) Nahraj tyto soubory do rootu repa
3) GitHub → Settings → Pages → Deploy from branch → main / root
4) Custom domain: energeticke-poradenstvi.cz
5) DNS:
   - A (@): 185.199.108.153, 185.199.109.153, 185.199.110.153, 185.199.111.153
   - CNAME (www): jakubholub-atlas.github.io (nebo tvoje GitHub Pages doména)
6) Enforce HTTPS zapnout (až se doména ověří)
